Welcome to HTTP Calculator Service
Usage Result
/serv/random/<Number> A random floating-point number in the range between 0 and <Number>. E.g., /serv/random/4.8 returns a random number in the range between 0 and 4.8.
/calc/add/<Number 1>/<Number 2> The sum of the two floating-point numbers <Number 1> and <Number 2>
/calc/sub/<Number 1>/<Number 2> The difference of the two floating-point numbers <Number 1> and <Number 2>
/calc/mul/<Number 1>/<Number 2> The product of the two floating-point numbers <Number 1> and <Number 2>
/calc/div/<Number 1>/<Number 2> The quotient of the two floating-point numbers <Number 1> and <Number 2>
/calc/mod/<Number 1>/<Number 2> The remainder when dividing the two integer numbers <Number 1> and <Number 2>
/calc/sqrt/<Number> The square root of the floating-point number <Number>
/calc/func/sin/<Number> The value of the sine function for the given floating-point radian angle <Number>
/calc/func/cos/<Number> The value of the cosine function for the given floating-point radian angle <Number>
/calc/func/tan/<Number> The value of the tangens function for the given floating-point radian angle <Number>
